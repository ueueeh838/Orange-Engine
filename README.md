##用法说明

### **1. 开发者用户**
对于开发者来说，想对接我们的橘子引擎(Orange Engine)，需要使用我们的对接库服务:
首先下载OrangeKernel.a静态库文件，并且进入我们的官网注册账号并申请一个开发者key，
根据使用例子填充你的开发者key即可使用。

### **2. 普通用户**
对于普通用户来说，想使用对接了橘子引擎的程序(我们的官方程序或第三方开发者的程序)，则需要下载
橘子引擎内核下载器，先下载自己对应内核版本的驱动，再运行程序即可。其中运行下载器需要一个用户key，
这个key需要通过telegram机器人 @Orangeqdbot 获取。一个telegram账号对应一个专属用户key，一个用户key
最多可以绑定五个设备，并且用户可以通过机器人自行解绑。
